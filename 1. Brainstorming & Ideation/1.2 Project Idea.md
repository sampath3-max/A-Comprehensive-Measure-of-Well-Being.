# Project Idea

## Project Title

A Comprehensive Measure of Well-Being

## Idea Overview

The Human Development Index (HDI) is a composite measure used to evaluate the development and well-being of people in different countries.

The project idea is to develop a Machine Learning-based application that predicts the HDI score of a country based on important socio-economic indicators.

The system uses the following indicators:

- Life Expectancy
- Mean Years of Schooling
- Expected Years of Schooling
- GNI Per Capita

These indicators are provided as input to a trained Machine Learning model. The model predicts the corresponding HDI score.

The predicted HDI score is then classified into one of the following categories:

- Low Human Development
- Medium Human Development
- High Human Development
- Very High Human Development

## Motivation

Traditional economic measures such as GDP do not provide a complete picture of human well-being.

A country may have strong economic growth but still face challenges related to education, health, or living standards.

Therefore, this project focuses on a more comprehensive measure of development by combining multiple socio-economic indicators.

## Proposed Technology

The project uses:

- Python for development
- Pandas for data processing
- Scikit-learn for Machine Learning
- Linear Regression for HDI prediction
- Flask for developing the web application
- HTML and CSS for the user interface
