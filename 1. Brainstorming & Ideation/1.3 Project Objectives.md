# Project Objectives

The primary objective of this project is to develop a Machine Learning-based system that predicts Human Development Index (HDI) scores using socio-economic indicators.

## Specific Objectives

1. To understand the relationship between socio-economic indicators and human development.

2. To use Life Expectancy as an indicator of the health dimension of human development.

3. To use Mean Years of Schooling and Expected Years of Schooling as indicators of the education dimension.

4. To use GNI Per Capita as an indicator of the standard of living.

5. To preprocess and prepare the dataset for Machine Learning.

6. To divide the dataset into training and testing data.

7. To train a Linear Regression model to predict HDI scores.

8. To evaluate the performance of the trained model.

9. To save the trained Machine Learning model for future use.

10. To develop a Flask-based web application for HDI prediction.

11. To allow users to enter socio-economic indicators through a web interface.

12. To display the predicted HDI score and its corresponding development category.

13. To create a simple and understandable system that demonstrates how Machine Learning can be applied to human development analysis.
