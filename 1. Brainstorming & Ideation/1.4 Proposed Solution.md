# Proposed Solution

## Overview

To address the problem of analyzing human development using multiple socio-economic indicators, a Machine Learning-based HDI prediction system is proposed.

The proposed system uses historical HDI data to train a Linear Regression model.

## Solution Workflow

The system follows the following workflow:

1. Collect the HDI dataset.

2. Identify relevant socio-economic indicators.

3. Select the input features:
   - Life Expectancy
   - Mean Years of Schooling
   - Expected Years of Schooling
   - GNI Per Capita

4. Select HDI Score as the target variable.

5. Divide the dataset into training and testing data.

6. Train a Linear Regression model using the training data.

7. Evaluate the model using testing data.

8. Save the trained model using Pickle.

9. Develop a Flask web application.

10. Allow users to enter socio-economic values.

11. Pass the values to the trained Machine Learning model.

12. Generate an HDI prediction.

13. Classify the predicted score as Low, Medium, High, or Very High.

14. Display the result to the user.
