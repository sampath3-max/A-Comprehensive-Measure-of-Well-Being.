# Use Cases

## Use Case 1: Predict HDI Score

### Actor

User

### Description

The user enters socio-economic indicators into the application to predict the Human Development Index (HDI) score.

### Input

The user provides:

- Life Expectancy
- Mean Years of Schooling
- Expected Years of Schooling
- GNI Per Capita

### Process

1. The user enters the required values.
2. The user submits the information.
3. The system processes the input values.
4. The Machine Learning model predicts the HDI score.

### Output

The system displays the predicted HDI score.

---

## Use Case 2: Classify Human Development Level

### Actor

System

### Description

The system classifies the predicted HDI score into a human development category.

### Categories

- Low Human Development
- Medium Human Development
- High Human Development
- Very High Human Development

### Output

The system displays the predicted HDI score along with its corresponding development category.

---

## Use Case 3: View Prediction Result

### Actor

User

### Description

The user views the final prediction generated by the system.

### Output

The system displays:

- Predicted HDI Score
- Human Development Category
