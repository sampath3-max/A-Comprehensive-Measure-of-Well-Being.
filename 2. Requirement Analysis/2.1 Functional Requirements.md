# Functional Requirements

Functional requirements describe what the system should do.

## FR-01: Accept User Input

The system shall allow the user to enter the following socio-economic indicators:

- Life Expectancy
- Mean Years of Schooling
- Expected Years of Schooling
- GNI Per Capita

## FR-02: Process Input Data

The system shall receive and process the values entered by the user.

## FR-03: Generate HDI Prediction

The system shall use the trained Machine Learning model to predict the Human Development Index (HDI) score.

## FR-04: Display Predicted HDI Score

The system shall display the predicted HDI score to the user.

## FR-05: Classify HDI Level

The system shall classify the predicted HDI score into the appropriate development category:

- Low Human Development
- Medium Human Development
- High Human Development
- Very High Human Development

## FR-06: Display Final Result

The system shall display the predicted HDI score and its corresponding human development category.
