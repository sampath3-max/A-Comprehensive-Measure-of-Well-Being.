# Hardware Requirements

The following hardware is sufficient to develop and run the HDI Prediction System.

## Processor

Intel Core i3 or equivalent processor and above.

## RAM

Minimum 4 GB RAM.

## Storage

At least 10 GB of available storage space.

## Display

A standard monitor or laptop display.

## Internet Connection

An internet connection may be required for:

- Downloading Python packages
- Accessing datasets
- Installing project dependencies
- Uploading the project to GitHub
