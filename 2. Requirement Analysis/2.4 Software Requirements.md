# Software Requirements

## Operating System

The application can run on:

- Windows
- Linux
- macOS

## Programming Language

- Python 3.x

## Libraries and Frameworks

The following libraries and frameworks are used:

- Pandas — for data processing
- NumPy — for numerical operations
- Scikit-learn — for Machine Learning
- Flask — for developing the web application

## Development Environment

The project can be developed using:

- Visual Studio Code
- PyCharm
- Jupyter Notebook

## Web Browser

A modern web browser is required to access the application, such as:

- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari

## Version Control

GitHub is used to store and manage the project source code and documentation.
