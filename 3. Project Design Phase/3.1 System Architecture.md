# System Architecture

The HDI Prediction System consists of multiple components that work together to collect user input, process the data, generate a prediction, and display the result.

## Main Components

### 1. User Interface

The user enters the required socio-economic indicators through a web-based input form.

The inputs are:

- Life Expectancy
- Mean Years of Schooling
- Expected Years of Schooling
- GNI Per Capita

### 2. Flask Application

The Flask application acts as the backend of the system.

It:

- Receives user input
- Processes the input values
- Sends the values to the trained Machine Learning model
- Receives the prediction
- Displays the result

### 3. Machine Learning Model

The trained Linear Regression model predicts the HDI Score based on the input socio-economic indicators.

The trained model is saved and reused by the Flask application.

### 4. HDI Classification

The predicted HDI score is classified into one of the following categories:

- Low
- Medium
- High
- Very High

### 5. Result Display

The final predicted HDI score and its classification are displayed to the user.
