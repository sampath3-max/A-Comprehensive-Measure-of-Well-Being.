# Project Flow

The HDI Prediction System follows a sequence of steps from dataset preparation to displaying the final prediction.

## Step 1: Dataset

The HDI dataset contains socio-economic indicators and their corresponding HDI scores.

## Step 2: Feature Selection

The following features are selected as inputs for the Machine Learning model:

- Life Expectancy
- Mean Years of Schooling
- Expected Years of Schooling
- GNI Per Capita

The target variable is:

- HDI Score

## Step 3: Train-Test Split

The dataset is divided into:

- 80% Training Data
- 20% Testing Data

The training data is used to train the model, while the testing data is used to evaluate its performance.

## Step 4: Model Training

A Linear Regression model is trained using the selected input features and the HDI Score.

## Step 5: Model Evaluation

The trained model is used to generate predictions using the testing data.

## Step 6: Model Saving

After training, the model is saved using Pickle so that it can be reused by the Flask application.

## Step 7: User Input

The user enters the required socio-economic indicators through the web interface.

## Step 8: HDI Prediction

The Flask application passes the user-provided values to the trained Machine Learning model.

## Step 9: HDI Classification

The predicted HDI score is classified as:

- Low
- Medium
- High
- Very High

## Step 10: Result Display

The predicted HDI score and its corresponding category are displayed to the user.
