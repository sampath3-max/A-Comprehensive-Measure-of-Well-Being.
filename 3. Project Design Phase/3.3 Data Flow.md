# Data Flow

The data flow describes how information moves through the HDI Prediction System.

## 1. User Input

The user enters the following socio-economic indicators:

- Life Expectancy
- Mean Years of Schooling
- Expected Years of Schooling
- GNI Per Capita

## 2. Input Processing

The Flask application receives the values submitted by the user.

The input values are converted into the required numerical format for the Machine Learning model.

## 3. Machine Learning Prediction

The processed input values are passed to the trained Linear Regression model.

The model uses the input features to predict the HDI Score.

## 4. HDI Classification

The predicted HDI Score is classified into a development category:

- Low
- Medium
- High
- Very High

## 5. Result Display

The final HDI Score and its classification are displayed to the user.
