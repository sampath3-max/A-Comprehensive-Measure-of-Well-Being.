# Machine Learning Workflow

The Machine Learning workflow describes the steps followed to train and use the HDI prediction model.

## Step 1: Load the Dataset

The HDI dataset is loaded using the Pandas library.

The dataset contains socio-economic indicators and the corresponding HDI Score.

## Step 2: Select Input Features

The following features are selected as input variables:

- Life Expectancy
- Mean Years of Schooling
- Expected Years of Schooling
- GNI Per Capita

## Step 3: Select Target Variable

The target variable used for prediction is:

- HDI Score

## Step 4: Split the Dataset

The dataset is divided into training and testing data.

- 80% of the data is used for training.
- 20% of the data is used for testing.

## Step 5: Select the Machine Learning Algorithm

Linear Regression is used because the system predicts a continuous numerical value, which is the HDI Score.

## Step 6: Train the Model

The Linear Regression model learns the relationship between the socio-economic indicators and the HDI Score using the training data.

## Step 7: Generate Predictions

The trained model generates HDI predictions for the testing data and new user inputs.

## Step 8: Save the Trained Model

The trained model is saved using Pickle so that it can be loaded and used by the Flask web application.
