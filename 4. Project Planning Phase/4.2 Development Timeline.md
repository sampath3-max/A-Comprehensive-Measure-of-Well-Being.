| Phase | Activities |
|---|---|

| Phase 2 | Requirement analysis |
| Phase 3 | System architecture and workflow design |
| Phase 4 | Project planning |
| Phase 5 | Environment setup and dependency installation |
| Phase 6 | Dataset preparation and understanding |
| Phase 7 | Machine Learning model development |
| Phase 8 | Model saving and integration with Flask |
| Phase 9 | Web application development |
| Phase 10 | Testing and validation |
| Phase 11 | Documentation |
| Phase 12 | Project demonstration |
