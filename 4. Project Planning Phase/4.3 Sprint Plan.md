# Sprint Plan

The project development is divided into multiple sprints. Each sprint focuses on a specific set of activities.

## Sprint 1: Project Setup

### Activities

- Set up the Python development environment.
- Install the required Python packages.
- Organize the project structure.
- Prepare the dataset.

### Expected Outcome

A ready-to-use development environment and project structure.

---

## Sprint 2: Dataset and Machine Learning Development

### Activities

- Load the HDI dataset.
- Understand the dataset structure.
- Select the required features.
- Divide the data into training and testing datasets.
- Train the Linear Regression model.

### Expected Outcome

A trained Machine Learning model capable of predicting HDI scores.

---

## Sprint 3: Model Saving and Integration

### Activities

- Evaluate the trained model.
- Save the trained model using Pickle.
- Load the saved model into the Flask application.

### Expected Outcome

A reusable Machine Learning model integrated with the backend application.

---

## Sprint 4: Web Application Development

### Activities

- Develop the Flask application.
- Create the input form.
- Accept user input.
- Connect the input form to the Machine Learning model.
- Display the prediction result.

### Expected Outcome

A working web application for HDI prediction.

---

## Sprint 5: Testing and Documentation

### Activities

- Test the Machine Learning model.
- Test valid and invalid inputs.
- Test the prediction workflow.
- Document the project.
- Prepare the project demonstration.

### Expected Outcome

A tested and documented HDI Prediction System.
