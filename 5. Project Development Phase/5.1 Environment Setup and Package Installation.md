# Environment Setup and Package Installation

## Objective

The first step in developing the HDI Prediction System is to prepare the Python development environment and install the required dependencies.

## Programming Language

The project is developed using:

- Python 3.x

## Required Packages

The following Python packages are required:

- Flask
- Pandas
- NumPy
- Scikit-learn

## Installing Dependencies

The required packages can be installed using the following command:

```bash
pip install -r requirements.txt
