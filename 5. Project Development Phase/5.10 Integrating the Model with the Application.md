# Integrating the Model with the Application

## Objective

The trained Machine Learning model is integrated with the Flask application to generate HDI predictions from user input.

## Receiving User Input

When the user submits the prediction form, the Flask application receives the input values.

The following values are collected:

- Life Expectancy
- Mean Years of Schooling
- Expected Years of Schooling
- GNI Per Capita

## Converting Input Values

The values received from the form are converted into numerical values before being passed to the Machine Learning model.
