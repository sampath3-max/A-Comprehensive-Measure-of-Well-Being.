# HDI Classification Logic

## Objective

After predicting the HDI Score, the system classifies the score into a corresponding human development category.

## Classification Rules

The predicted HDI Score is classified using the following ranges:

| HDI Score Range | Category |
|---|---|
| Below 0.550 | Low Human Development |
| 0.550 – 0.699 | Medium Human Development |
| 0.700 – 0.799 | High Human Development |
| 0.800 and above | Very High Human Development |

## Classification Process

```text
Predicted HDI Score
        ↓
Is Score < 0.550?
        ↓
   Low HDI
        │
        └── If No
              ↓
     Is Score < 0.700?
              ↓
        Medium HDI
              │
              └── If No
                    ↓
           Is Score < 0.800?
                    ↓
              High HDI
                    │
                    └── If No
                          ↓
                    Very High HDI
