# Technical Documentation

## 1. System Overview

The HDI Prediction System is a Machine Learning-based web application that predicts the Human Development Index (HDI) Score using socio-economic indicators.

The system consists of:

- A dataset.
- A Machine Learning model.
- A Flask backend application.
- An HTML-based user interface.

## 2. Input Features

The system uses the following input features:

- Life Expectancy
- Mean Years of Schooling
- Expected Years of Schooling
- GNI Per Capita

The target variable used by the Machine Learning model is:

HDI Score
