# A Comprehensive Measure of Well-Being

## HDI Prediction Using Machine Learning

This project uses Machine Learning to predict the Human Development Index (HDI) based on socio-economic indicators such as:

- Life Expectancy
- Mean Years of Schooling
- Expected Years of Schooling
- GNI Per Capita

The system predicts an HDI score and classifies the result into Low, Medium, High, or Very High human development.
